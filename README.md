# hospital-moscati-smartmenu

QR-based multilingual digital menu system for food services inside Hospital Moscati.

## Overview
Moscati SmartMenu is a web-based application that allows patients, visitors, and staff to access restaurant menus by scanning QR codes placed throughout the hospital.

The system supports multiple restaurants, bilingual content (Spanish and English), and an administrator panel for managing menus, prices, and availability in real time.

## Key Features
- QR-based access (no app download required)
- Multilingual support (ES / EN)
- Multiple restaurants and cafeterias
- Context-aware QR routing
- Admin dashboard for menu management
- Mobile-first responsive design

## Tech Stack (Planned)
- Frontend: Next.js (React)
- Backend: Next.js API Routes
- Database: Supabase (PostgreSQL)
- Auth: Supabase Auth
- Styling: Tailwind CSS
- Hosting: Vercel

# Moscati SmartMenu

Sistema de menú digital multilingüe basado en QR para los servicios de alimentación del Hospital Moscati.

## Descripción general
Moscati SmartMenu es una aplicación web que permite a pacientes, visitantes y personal acceder a los menús de los restaurantes escaneando códigos QR distribuidos por todo el hospital.

El sistema admite múltiples restaurantes, contenido bilingüe (español e inglés) y un panel de administración para gestionar menús, precios y disponibilidad en tiempo real.

## Características principales
- Acceso basado en QR (sin necesidad de descargar la app)
- Soporte multilingüe (ES/EN)
- Múltiples restaurantes y cafeterías
- Enrutamiento de QR contextual
- Panel de administración para la gestión del menú
- Diseño responsivo para móviles

## Pila tecnológica (en desarrollo)
- Frontend: Next.js (React)
- Backend: Rutas API de Next.js
- Base de datos: Supabase (PostgreSQL)
- Autorización: Supabase Auth
- Estilo: Tailwind CSS
- Alojamiento: Vercel
